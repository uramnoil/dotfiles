# dotfiles

## Install

### for macOS

#### hobby

```sh
xcode-select --install
git clone https://github.com/uramnoil/dotfiles.git
cd dotfiles
make macos_hobby
```

#### business

```sh
xcode-select --install
git clone https://github.com/uramnoil/dotfiles.git
cd dotfiles
make macos_biz
```

### for Linux

```sh
sudo apt install -y git
git clone https://github.com/uramnoil/dotfiles.git
cd dotfiles
make linux
```

### for Linux Desktop

```sh
sudo apt install -y git
git clone https://github.com/uramnoil/dotfiles.git
cd dotfiles
make linux
```
